## Prerequisites

To install and use this library, your project must meet the following **minimum requirements**:

- **Angular**: This library is compatible only with **Angular 18 or higher**.

  | Dependency        | Minimum Required Version |
  | ----------------- | ------------------------ |
  | `@angular/common` | `^18.2.0`                |
  | `@angular/core`   | `^18.2.0`                |
  | `rxjs`            | `^7.8.0`                 |

<Alert type="info">
  You have a [blank Angular
  project](https://github.com/rperezll/ng-fast-toast/tree/main/test/angular-project)
  with the **ng-fast-toast** library installed, in case you want to run it and
  verify its functionality.
</Alert>

If the listed prerequisites are met, we can proceed with the installation of the library.

## Install the library

To integrate toast notifications into your Angular project, you need to install the `ng-fast-toast` library. This package provides an _easy-to-use_ service for displaying customizable toast notifications.

Run the following command to install it:

```bash
npm i ng-fast-toast@latest
```

<Alert type="info">
  If you want more information about the npm package, check out its [official
  npm page](https://www.npmjs.com/package/ng-fast-toast).
</Alert>
